# MedLex V-index

의학 용어(Medical Lexicon)의 접두사(Prefix)와 접미사(Suffix)를 체계적으로 색인, 검색 및 관리할 수 있도록 설계된 데이터 분석 및 학습용 단일 파일 웹 애플리케이션입니다.

## 핵심 기능 명세 (Key Features)

### 1. 의학 용어 접사 검색 및 필터링 시스템
* **분류별 색인 탐색**: 의학 용어 구성의 핵심인 접두사(Prefix)와 접미사(Suffix)를 데이터 성격에 따라 시각적으로 구분하여 탐색할 수 있습니다.
* **실시간 텍스트 스크리닝**: 검색창 입력 값에 동기화되어 전체 데이터베이스 내 일치하는 단어와 의미를 지연 없이 실시간으로 필터링합니다.

### 2. 카테고리 기반 데이터 관리 (Folder System)
* **폴더별 분류**: 의학 계통별 또는 학습 목적별로 단어를 분류하여 관리할 수 있는 폴더 구조를 제공합니다.
* **동적 셀렉트 박스 연동**: 사용자가 정의하거나 생성한 폴더 목록이 UI 내의 선택 셀렉터에 실시간으로 반영 및 갱신됩니다.

### 3. 사용자 커스텀 데이터 확장 (CRUD)
* **직접 입력 기능 (UX-13)**: 사전 정의된 데이터 외에 사용자가 필요한 의학 용어 및 뜻을 직접 입력하여 데이터베이스를 확장할 수 있습니다.
* **데이터 정형성 검증**: 필수 값 누락이나 중복 입력을 방지하기 위한 유효성 검증(Validation) 및 전용 다이얼로그 시스템이 구축되어 있습니다.

### 4. 실시간 통계 분석 대시보드
* **탭 기반 데이터 분석**: 별도로 분리된 분석 탭을 통해 현재 저장된 데이터의 전체 통계, 접두사/접미사 비율, 폴더별 분포 등을 시각적으로 확인 가능합니다.
* **상태 표시 배지**: 주요 통계 수치나 바로가기 항목을 배지(Badge) 형태로 UI에 배치하여 현재 데이터의 누적 상태를 직관적으로 파악할 수 있도록 돕습니다.

### 5. 사용자 로깅 및 영구 저장 시스템
* **텍스트 로깅 기능**: 학습 중 메모나 오답 노트, 피드백 등을 기록할 수 있는 인앱 로깅 시스템을 제공합니다.
* **타임스탬프 기반 최신순 정렬**: 기록된 데이터는 연월일 및 시분(YYYY.MM.DD HH:MM) 형태의 타임스탬프와 함께 최신순으로 상단에 자동 정렬됩니다.
* **로컬 스토리지 연동**: 입력된 모든 커스텀 데이터와 로그는 브라우저의 로컬 스토리지(Local Storage)에 저장되어 페이지를 새로고침하거나 브라우저를 종료해도 안전하게 보존됩니다.

## 기술적 스펙 (Technical Specifications)

* **아키텍처 (Architecture)**: 외부 라이브러리나 프레임워크(React, Vue 등) 의존성이 전혀 없는 No-Dependency 기반의 순수 HTML5, CSS3, Vanilla JavaScript(ES6+) 구조입니다. 단일 파일(Single File) 내에서 상태 기반 UI 렌더링이 이루어집니다.
* **UI/UX 디자인**: 미디어 쿼리를 통한 반응형 웹 디자인(Responsive Web Design)이 적용되어 있으며, 모바일 환경에서의 오작동을 방지하기 위해 뷰포트 확대/축소가 고정되어 있습니다. CSS 변수(CSS Variables) 체계를 기반으로 설계되어 전역 라이트/다크 테마 전환이 가능합니다.
* **성능 최적화 (Performance)**: 모바일 및 저사양 기기에서의 부드러운 구동을 위해 렌더링 부하가 큰 웹 필터 연산을 배제하고, 브라우저의 GPU 가속 레이어를 활용하는 transform 속성 위주로 애니메이션과 인터랙션을 제어합니다.
